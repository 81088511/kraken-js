Getting started with Kraken

Setting Up

The fastest way to get started with Kraken is to use the Yeoman generator for it. Start by installing it globally using npm: sudo npm install -g generator-kraken

Once installed, you can create a basic project using the generator. Type yo kraken and follow the prompts; you will be asked for a project name, description and author name.

Configuring Kraken

Let's say you want to create a simple application. Typically, most of the action would happen in your index.js file. You'd load up your required middleware, configure the server, create all the routes and maybe even throw in some inline logic because, hey, it's only a couple of lines, right? As your application grows, this becomes unmanageable and messy. Kraken helps you stay organized by imposing a sound structure and strategy.

Let's inspect a basic Kraken project to understand its structure. First let's look at our basic index.js entry point. You will very quickly notice that its very small. Where's all the configuration? Where are the routes? 

'use strict';

var kraken = require('kraken-js'),
    app = {};


app.configure = function (nconf, next) {
    // Fired when an app configures itself
    next(null);
};


app.requestStart = function (server) {
    // Fired at the beginning of an incoming request
};


app.requestBeforeRoute = function (server) {
    // Fired before routing occurs
};


kraken.create(app).listen(function (err) {
    if (err) {
        console.error(err);
    }
});
